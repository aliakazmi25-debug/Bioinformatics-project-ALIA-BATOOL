This repository contains a comprehensive bioinformatics analysis of a genomic sequence, organized according to the required project template. It includes DNA analysis with BLAST results and ORF identification, RNA secondary structure prediction, and detailed protein characterization including domain analysis and physicochemical properties. Additionally, the project features sub-cellular localization results and a phylogenetic study using the Neighbor-Joining method to explore evolutionary relationships.
